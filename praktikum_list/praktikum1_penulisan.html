Praktikum 1 - Penulisan CSS

1. Inline CSS

Paragraf ini menggunakan inline CSS langsung pada atribut style. 

<h1 style="color: red">Judul Web</h1>

2. Internal CSS

Paragraf ini menggunakan internal CSS yang ditulis di dalam tag <style> pada bagian <head>. 

<style>
  h1 { color: red; }
</style>

3. External CSS

External CSS ditulis di file terpisah berekstensi .css lalu dipanggil di dalam tag head. 

<link rel="stylesheet" href="namafile.css">
